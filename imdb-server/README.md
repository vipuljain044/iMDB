# IMDb API
Demo - https://app-imdb-api.herokuapp.com/api-docs/

This NodeJs project is a simplified backend-end of the IMDb website.

### Stack used:
- NodeJs
- Mongoonse
- Express
- jsonwebtoken

### Features:
- User can perform registeration and login
- Admin can Add/Edit/Delete Movies  

#### Steps to run 
    npm install
    npm run start
